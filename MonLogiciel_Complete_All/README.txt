MON LOGICIEL - COMPLETE ALL
===========================
Contenu:
- MonLogicielDesktop/   -> WPF (.NET 7) projet avec modules Pub, Cuisine, CRM, Stock, Factures
- MonLogicielWeb/       -> ASP.NET Core (net7) project with API + pages front-end
- data/catalog.json     -> exemples (panneaux, cuisines, clients)
Instructions:
1) Ouvrir MonLogiciel_Complete_All dans l\'Explorateur.
2) Ouvrir chaque .csproj dans Visual Studio 2022/2023 (restaurer NuGet si demandé).
3) Lancer WPF (MonLogicielDesktop) ou Web (MonLogicielWeb).
